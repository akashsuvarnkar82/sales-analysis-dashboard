# 📊 Sales Data Analysis and Visualization

## 📌 Project Overview

This project focuses on analyzing sales data using statistical techniques and visualizing insights through an interactive Power BI dashboard. The goal is to understand sales patterns, identify key drivers of revenue, and support data-driven decision-making.

---

## 🎯 Objectives

* Analyze sales distribution and trends
* Evaluate performance across geographies and salespersons
* Study the relationship between units sold and revenue
* Apply statistical methods (Hypothesis Testing & Confidence Interval)
* Build an interactive Power BI dashboard

---

## 📂 Dataset

The dataset (**awesome.csv**) contains the following columns:

* **Sales Person**
* **Geography**
* **Product**
* **Amount**
* **Units**

---

## 🛠️ Tools & Technologies Used

* Python (Pandas, NumPy, Matplotlib, Seaborn, SciPy)
* Microsoft Power BI
* Jupyter Notebook

---

## 📊 Key Analysis Performed

### 🔹 Exploratory Data Analysis (EDA)

* Distribution analysis of Amount
* Geography-wise and Salesperson performance
* Outlier detection using boxplots

### 🔹 Bivariate Analysis

* Relationship between Units and Amount
* Correlation analysis

### 🔹 Statistical Analysis

* **Hypothesis Testing (T-test)**
  → Compared sales between USA and UK

* **Confidence Interval (95%)**
  → Estimated range of average sales

---

## 📈 Key Insights

* Sales are positively correlated with units sold
* Revenue is influenced by a few high-value transactions
* Sales performance varies among salespersons
* No statistically significant difference between USA and UK sales
* Presence of outliers indicates bulk or high-value orders

---

## 📊 Dashboard

An interactive Power BI dashboard was created to visualize:

* Total Sales, Units, and Average Sales
* Sales by Geography and Sales Person
* Units vs Amount relationship
* Product-wise sales distribution
